# MK-Macropad
Wireless 9‑key customizable macropad using Bluetooth. Sends commands and macros to your PC, features dynamic LED animations, and powers on/off automatically. Fully editable firmware and user interface, designed for makers who want a flexible and hackable device.


EN:

MK Macropad is an open‑source ESP32‑based macropad featuring:

- a fully customizable firmware
- a modern user interface
- a flexible layout and LED effects system
- an architecture designed for easy modification

## 🎯 Project Goals
- provide a simple, powerful and modifiable macropad
- encourage makers to create their own layouts, themes and mods
- ensure that all improvements remain open‑source
