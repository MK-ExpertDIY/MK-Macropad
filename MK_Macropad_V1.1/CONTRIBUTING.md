# Contributing to MK Macropad

Thank you for your interest in contributing to MK Macropad!

## 🧩 Ways to Contribute
- fix a bug
- add a layout
- create a new LED effect
- improve the user interface
- optimize the firmware
- propose a new feature
- improve documentation

## 📤 Submitting a Pull Request
1. Fork the repository  
2. Create a branch: `feature/my-feature`  
3. Make your changes  
4. Add clear comments in the code  
5. Submit a PR using the provided template  

## 🛡 License
By contributing, you agree that your contribution will be released under the **GPL‑3.0** license.
